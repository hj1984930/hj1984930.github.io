## 模拟股指期货报撤单数据

### 开始

* 因项目需要
* 股指期货的报撤单数据属于绝对机密，中金所不肯提供
* 那就只能模拟造出来了

### 逻辑及思路

* 20万客户，每天3万交易客户
* 每天20万笔报撤单记录
* 一年大概五千万条数据
* 需要模拟的数据          
* 交易日  tradingday : date
* 客户id clientid : int
* 合约id instrumentid : text
* 买卖方向 direction : text
* 报单日期 insertdate : date
* 报单时间 inserttime : time

### Greenplum创建表

~~~sql
create table if(
tradingday date,
clientid int,
instrumentid char(6),
direction char(1),
insertdate date,
inserttime time
)
WITH (appendonly=true,orientation=column,compresstype=QUICKLZ,COMPRESSLEVEL=1)  
distributed by (clientid)
~~~
